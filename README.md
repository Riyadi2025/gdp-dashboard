# AI Website Builder

A powerful, AI-driven website builder that creates stunning, professional websites in minutes. Just describe your business and get a complete website with content, images, and responsive design - all generated by AI.

## ✨ Features

### 🤖 AI-Powered Generation
- **Smart Content Creation**: AI generates compelling copy, headlines, and descriptions
- **Dynamic Image Generation**: Automatic image creation with DALL-E integration
- **Intelligent Section Layout**: AI determines optimal page structure based on business type
- **SEO Optimization**: Built-in SEO best practices and meta tag generation

### 🎨 Professional Design
- **Responsive Design**: Automatically optimized for desktop, tablet, and mobile
- **Modern Templates**: Choose from business, portfolio, e-commerce, and more
- **Custom Themes**: AI-generated color schemes and typography
- **Drag-and-Drop Editor**: Visual editing with real-time preview

### 🚀 Quick Deployment
- **One-Click Publishing**: Instant website deployment
- **Export Options**: Download as HTML, React, or other formats
- **Custom Domains**: Connect your own domain name
- **Free Hosting**: Built-in hosting solution

### 📊 Business Features
- **Analytics Integration**: Google Analytics, Facebook Pixel, and more
- **Contact Forms**: Automatic form generation and processing
- **Blog System**: Built-in blogging capabilities
- **E-commerce Ready**: Product catalogs and shopping cart functionality

## 🛠️ Technology Stack

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS, HeadlessUI
- **AI Integration**: OpenAI GPT-4, DALL-E 3
- **Database**: Local JSON storage (extensible to MongoDB/PostgreSQL)
- **Authentication**: NextAuth.js (optional)
- **Deployment**: Vercel-ready

## 🚀 Quick Start

### Prerequisites

- Node.js 18.0 or higher
- npm or yarn package manager
- OpenAI API key

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/ai-website-builder.git
cd ai-website-builder
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
```

3. **Set up environment variables**
```bash
cp .env.example .env.local
```

Edit `.env.local` with your configuration:
```env
# OpenAI API Configuration
OPENAI_API_KEY=your_openai_api_key_here

# Next.js Configuration
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=your_nextauth_secret_here

# Application Configuration
APP_URL=http://localhost:3000
APP_NAME=AI Website Builder
```

4. **Run the development server**
```bash
npm run dev
# or
yarn dev
```

5. **Open your browser**
Navigate to [http://localhost:3000](http://localhost:3000)

## 📖 Usage Guide

### Creating Your First Website

1. **Describe Your Business**
   - Enter a detailed description of your business
   - Select a template type (business, portfolio, e-commerce, etc.)
   - Choose your preferred tone and style

2. **AI Generation**
   - Click "Generate My Website"
   - AI creates complete website structure
   - Content, images, and design are automatically generated

3. **Customize and Edit**
   - Use the visual editor to modify content
   - Adjust colors, fonts, and layout
   - Add or remove sections as needed

4. **Publish or Export**
   - One-click publishing to live URL
   - Export as HTML for self-hosting

## 🔧 Configuration

### OpenAI Configuration

The system uses OpenAI's GPT-4 for content generation and DALL-E 3 for images. Configure in `lib/openai.ts`:

```typescript
const openai = new OpenAI({
  apiKey: process.env.OPENAI_API_KEY,
});
```

### Customizing AI Prompts

Modify the AI prompts in `lib/openai.ts` to adjust the generation behavior:

```typescript
private static buildWebsiteSystemPrompt(context?: AIGenerationContext): string {
  const basePrompt = `You are an expert web designer and content creator...`;
  // Customize the prompt based on your needs
}
```

### Adding New Section Types

1. **Add to Types**
```typescript
// types/index.ts
export type SectionType = 
  | 'hero'
  | 'about'
  | 'services'
  | 'your-new-section'; // Add your new section type
```

2. **Update AI Prompt**
```typescript
// lib/openai.ts
const sectionPrompts = {
  'your-new-section': 'Create a custom section with specific requirements...',
};
```

3. **Add Rendering Logic**
```typescript
// pages/editor.tsx
const renderSection = (section: any, index: number) => {
  switch (type) {
    case 'your-new-section':
      return <YourNewSectionComponent {...section} />;
    // ... other cases
  }
};
```

## 🔌 API Reference

### Generate Website
```typescript
POST /api/generate/website
{
  "prompt": "Business description",
  "context": {
    "businessType": "agency",
    "industry": "technology",
    "tone": "professional",
    "style": "modern"
  }
}
```

### Generate Section
```typescript
POST /api/generate/section
{
  "type": "hero",
  "prompt": "Create a hero section for a fitness studio",
  "context": {
    "businessType": "fitness",
    "targetAudience": "health-conscious adults"
  }
}
```

### Generate Image
```typescript
POST /api/generate/image
{
  "prompt": "Modern office space with natural lighting",
  "size": "1024x1024"
}
```

## 🎨 Customization

### Theme Configuration

Customize the default theme in `utils/index.ts`:

```typescript
export const getDefaultGlobalStyles = () => ({
  colors: {
    primary: '#3b82f6',
    secondary: '#64748b',
    accent: '#06b6d4',
    // ... customize colors
  },
  typography: {
    fontFamily: 'Inter, sans-serif',
    // ... customize fonts
  },
});
```

### Adding Custom Components

1. **Create Component**
```typescript
// components/sections/CustomSection.tsx
export const CustomSection = ({ content, styles }) => {
  return (
    <section style={styles}>
      <div className="container">
        <h2>{content.heading}</h2>
        <p>{content.description}</p>
      </div>
    </section>
  );
};
```

2. **Register Component**
```typescript
// components/sections/index.ts
export { CustomSection } from './CustomSection';
```

3. **Add to Editor**
```typescript
// pages/editor.tsx
import { CustomSection } from '@/components/sections';

const renderSection = (section: any, index: number) => {
  switch (section.type) {
    case 'custom':
      return <CustomSection {...section} />;
    // ... other cases
  }
};
```

## 🚀 Deployment

### Vercel Deployment

1. **Connect Repository**
   - Push code to GitHub
   - Import project in Vercel dashboard

2. **Configure Environment Variables**
   - Add `OPENAI_API_KEY` in Vercel dashboard
   - Set `NEXTAUTH_URL` to your production domain

3. **Deploy**
   - Automatic deployment on every push
   - Custom domain configuration available

### Self-Hosting

1. **Build Production Version**
```bash
npm run build
npm run start
```

2. **Docker Deployment**
```dockerfile
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm ci --only=production
COPY . .
RUN npm run build
EXPOSE 3000
CMD ["npm", "start"]
```

## 🐛 Troubleshooting

### Common Issues

**OpenAI API Errors**
- Verify API key is correct
- Check API quota and billing
- Ensure proper model permissions

**Build Failures**
- Clear `.next` folder and `node_modules`
- Reinstall dependencies
- Check Node.js version compatibility

**Runtime Errors**
- Check browser console for errors
- Verify environment variables
- Review server logs

**Performance Issues**
- Optimize image sizes
- Reduce API calls
- Implement caching strategies
